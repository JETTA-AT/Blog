Spark函数:浅析SparkSession

Posted on 2019-02-18 | In Spark |

在理解spark-session之前让我们理解入口点，一个入口点是控制从操作系统传递到提供的程序的地方。
在2.0入口之前，spark-core是sparkContext.Apache Spark是一个功能强大的集群计算引擎，因此它专为快速计算大数据而设计。
而这个入口可就是SparkContext。

SparkContext在Apache Spark中的功能：

1.获取spark应用程序的当前状态
2.设置配置
3.访问各种服务
4.取消job
5.取消一个stage
6.关闭清洁
7.注册Spark-Listener
8.可编程动态分配
9.访问持久性RDD

SparkConf是创建spark上下文对象所必需的，它存储配置参数，如appName（用于标识spark驱动程序），core的数目和在工作节点上运行的执行程序的内存大小。

val conf = new SparkConf()
    //core数目 local 本地单核/ local[2] 本地2核 / local[*]  本地全部核心
  .setMaster("local")  
  .setAppName("Spark Practice")
val sc = new SparkContext(conf)

缺点

随着功能模块的不断增加，而为了使用SQL，Hive和Streaming，需要创建单独的上下文。

创建StreamingContext例子

val conf = new SparkConf()
  .setMaster("local[*]")
  .setAppName("NetworkWordCount")
  .set("spark.io.compression.codec","snappy")
val ssc = new StreamingContext(conf, Seconds(args(2).toInt))

SparkSession - 新入口点

众所周知，在以前的版本中，sparkcontext 是spark的入口点，因为RDD是主要的API，它是使用上下文API创建和操作的。 对于每个其他API，我们需要使用不同的context。

对于流式传输，我们需要streamingContext。 对于SQL sqlContext和hive hiveContext.，因为dataSet和DataFrame API正在成为新的独立API，我们需要为它们构建入口点。 因此在spark 2.0中，我们为DataSet和DataFrame API创建了一个新的入口点构建，称为Spark-Session。

创建一个SparkSession入口

val spark = SparkSession.builder()
  .master("local")
  .appName("example of SparkSession")
  .config("spark.some.config.option", "some-value")
  .getOrCreate()

完整的代码

import org.apache.spark.sql.{Row, SparkSession}
import org.apache.spark.sql.types._

object test_SparkSession {
  def main(args: Array[String]): Unit = {
    val spark: SparkSession = SparkSession.builder
      .appName("My Spark Application")  // optional and will be autogenerated if not specified
      .master("local[*]")               // avoid hardcoding the deployment environment
      .config("spark.io.compression.codec","snappy")
      .getOrCreate

    //使用SparkSession创建DataFrame，并执行spark-SQL
    val customSchema = StructType(
      List(
        StructField("creative", StringType, true),
        StructField("uv", IntegerType, true)
      )
    )
    val df = spark.read.format("com.databricks.spark.csv")
      .schema(customSchema)                            //绑定Schema
      .load(args(0))
      .registerTempTable("creative_id_all_2")        //将数据转换成零时表

    val SQL = spark.sql("select * from creative_id_all_2 order by uv desc limit 10")
      .show()


    //创建自定义格式
    val schemaString = "name,age"
    val fields = schemaString.split(",").map(fieldName => StructField(fieldName, StringType, nullable = true))
    val schema = StructType(fields)
    val personRDD = spark.sparkContext.textFile(args(0))
    val rowRDD = personRDD.map(_.split(",")).map(attr => Row(attr(0),attr(1).trim()))
    val personDF = spark.createDataFrame(rowRDD,schema).show()


    //原有的sparkContext被包在SparkSession里面
    val file=spark.sparkContext.textFile(args(2))

    val word=file.flatMap(lines=>lines.split(" "))
      .map(word=>(word,1)).reduceByKey(_+_)
    word.foreach(println)
  }
}
